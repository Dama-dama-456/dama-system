# دليل نظام داما القابضة للإدارة

---

## نظرة عامة على النظام

نظام داما القابضة للإدارة هو منصة ويب متكاملة تُتيح إدارة كافة الموارد البشرية والتشغيلية للمجموعة، وتشمل: الموظفين، المستشارين، المتدربين، الشركات، المنشآت غير الربحية، الخدمات، والمشاريع.

---

## المكونات التقنية

| المكوّن | التقنية |
|---|---|
| الواجهة الأمامية | React + TypeScript (واجهة عربية RTL) |
| الخادم الخلفي | Node.js + Express + TypeScript |
| قاعدة البيانات | PostgreSQL (مدمجة في المشروع) |
| المصادقة | JWT (رمز مشفر صالح 7 أيام) |

---

## الأدوار والصلاحيات

| الدور | العرض | الإضافة والتعديل | الحذف | إدارة المستخدمين |
|---|---|---|---|---|
| **admin** (مدير) | ✅ | ✅ | ✅ | ✅ |
| **manager** (مشرف) | ✅ | ✅ | ❌ | ❌ |
| **viewer** (مشاهد) | ✅ | ❌ | ❌ | ❌ |

---

## سير العمل — تسجيل الدخول

```
1. يفتح المستخدم الموقع
2. يُدخل اسم المستخدم وكلمة المرور
3. يتحقق الخادم من البيانات في جدول users
4. عند النجاح: يُرسل رمز JWT صالح 7 أيام
5. يُخزَّن الرمز في المتصفح (localStorage)
6. كل طلب لاحق يحمل الرمز في Header: Authorization: Bearer <token>
7. عند انتهاء الجلسة: يُحذف الرمز ويُعاد التوجيه لصفحة الدخول
```

**حساب المدير الافتراضي:**
- اسم المستخدم: `admin`
- كلمة المرور: `Admin@123`

> تغيير كلمة المرور ممكن من صفحة إدارة المستخدمين

## بنية قاعدة البيانات

```
users          — المستخدمون والصلاحيات
employees      — بيانات الموظفين
consultants    — بيانات المستشارين
trainees       — بيانات المتدربين
companies      — بيانات الشركات
nonprofits     — بيانات المنشآت غير الربحية
services       — بيانات الخدمات
projects       — بيانات المشاريع (مرتبطة بالشركات والخدمات)
counters       — عدادات ترقيم تلقائي (EMP-xxxx، CON-xxxx)
```

**ملاحظة مهمة:** جميع الجداول تستخدم الحذف الناعم (is_deleted = true)
بمعنى: البيانات المحذوفة لا تُمحى فعلياً من قاعدة البيانات بل تُخفى فقط

---

## خطوات البدء السريع

```
1. افتح الموقع
2. سجّل الدخول بـ: admin / Admin@123
3. غيّر كلمة المرور من إدارة المستخدمين
4. أضف الشركات والخدمات أولاً (لأن المشاريع تحتاجها)
5. أضف الموظفين والمستشارين والمتدربين
6. أنشئ المشاريع وربطها بالشركات والخدمات
7. أضف مستخدمين جديدين حسب الحاجة وحدد صلاحياتهم
```
